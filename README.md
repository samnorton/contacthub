#  contacthub

A simple app to manage your contacts online.

![](https://i.imgur.com/X3FK6Ke.png)


## Installation

```
git clone git@github.com:samnorton/contacthub.git
cd contacthub
rails db:create
rails db:migrate
rails s
open localhost:3000
```

## App Preview

Visit https://contacthub.herokuapp.com/ for a preview of the app.
